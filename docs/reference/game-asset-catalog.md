## Key Differences: QL vs Q3A

- **QL has 8,968 files** vs Q3A's 3,597 — QL is ~2x larger
- **QL-exclusive maps (149):** added for CA, duel, FFA game modes
- **Q3A-exclusive maps (36):** original Q3 content, some not ported to QL
- **Shared maps (0):** the classic Q3 arenas present in both
- **Texture formats:** QL uses `.png` extensively; Q3A uses `.tga` and `.jpg` only
- **QL adds:** `fonts/` directory with TrueType fonts for HUD rendering
- **QL missing:** `video/` directory (Q3A had cinematic ROQ video files)
- **Sound:** QL has `.ogg` tracks; Q3A uses `.wav` only for most sounds
