Drawing spec
Consistency rules
Grid — 24×24 viewBox. All endpoints on 2px grid. No half-pixels.
Stroke — 1.5px default, 1.25px for dim/secondary detail, 2px for emphasis lines (rail shot, seam axis). stroke-linecap: square, stroke-linejoin: miter. Brutalist: no rounded ends.
Fill — Use sparingly. Solid gold fill signals present / active / hit. Hollow stroke signals available / idle.
Color — Default currentColor. Status icons (35–37) lock their color. All others tintable via parent CSS color.
Naming — icon-[group]-[id], e.g. icon-wpn-rail, icon-evt-exc.
Delivery — Ship as single sprite (icons.svg with <symbol>) + individual files. Use <svg><use href="#icon-xxx"/></svg>.
